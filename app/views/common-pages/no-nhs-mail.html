Register to use the apply for a maternity exemption certificate service

You must have an NHSmail account to use this service.

If you do not have an NHSmail account

If you're working in a trust or organisation that uses NHSmail, you can contact your Local Administrator or IT Helpdesk to set up an account.

For more information on joining NHSmail visit the NHSmail help page.

To register to use this service

Email nhsbsa.healthcosts@nhs.net

We'll ask for your:

  • name
  • NMC/GMC number
  • NHSmail address
  • UK telephone number
  • primary care organisation

This should take about 5 minutes.
